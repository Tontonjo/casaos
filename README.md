## Access: 
Once installed, you can access casa os using the hostname of your host: if it's "casaos" ->
http://casaos

## Default passwords:
| OS | Service | username | password |  
| ----------- | ----------- | ----------- | ----------- |
| ISO debian | SSH | root  | password |

To change the default ssh root "password"
```shell
passwd root
```
